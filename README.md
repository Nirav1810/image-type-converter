A modern, fast, and secure image conversion tool built with Next.js and Tailwind CSS. Convert images entirely in your browser without uploading files to any server.

## Features ‚ú®

- **Client-Side Processing:** All conversions happen locally in your browser for maximum privacy.
- **Drag & Drop Interface:** Simple and intuitive file upload experience.
- **Support for Major Formats:** Convert between JPEG, PNG, and WEBP.
- **Customizable Quality:** Adjust quality for JPEG and WEBP formats to balance size and fidelity.
- **Image Scaling:** Resize PNG images to reduce file size effectively.
- **Batch Conversion:** Convert multiple files at once.
- **Zip Download:** Download all converted images in a single click (if multiple files).
- **Responsive Design:** Optimized for desktop and mobile devices.

## Getting Started üöÄ

1.  Clone the repository:
    ```bash
    git clone https://github.com/Nirav1810/image-type-converter.git
    cd image-type-converter
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Run the development server:
    ```bash
    npm run dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Usage üí°

1.  Drag and drop your image files onto the upload area.
2.  Select your desired output format (WEBP, JPEG, or PNG).
3.  Adjust the **Quality** (for JPEG/WEBP) or **Scale** (for PNG) settings.
4.  Click **Convert Images**.
5.  Download individual files or use **Download All**.
